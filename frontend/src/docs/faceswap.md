# 视频换脸功能使用说明

## 功能介绍

视频换脸功能可以将视频中的人脸替换为您上传的人脸照片，支持多人脸同时替换。

## 使用流程

### 第一步：上传视频

1. 点击上传区域或拖拽视频文件到上传区域
2. 支持的视频格式：MP4、WebM
3. 上传过程中会显示进度条
4. 上传完成后视频会自动播放预览

### 第二步：检测人脸

1. 视频上传完成后，系统会自动开始检测人脸
2. 也可以手动点击「检测人脸」按钮重新检测
3. 检测完成后，视频中的人脸会用绿色方框标记
4. 每个人脸都有一个编号标签

### 第三步：选择要替换的人脸

1. 点击视频中的人脸方框或右侧面板中的人脸卡片
2. 选中的人脸会用蓝色边框高亮显示
3. 可以选择多个人脸进行替换
4. 再次点击可以取消选择

### 第四步：上传替换人脸

1. 为每个选中的人脸上传一张替换照片
2. 点击「上传新脸」按钮选择照片
3. 建议使用正面、清晰的人脸照片
4. 支持的图片格式：JPG、PNG、WebP

### 第五步：开始换脸

1. 确认所有选中的人脸都已上传替换照片
2. 可选：勾选「HD 人脸增强」获得更好的效果（消耗更多积分）
3. 点击「开始换脸」按钮
4. 等待处理完成，通常需要 1-3 分钟

### 第六步：下载结果

1. 处理完成后会弹出下载窗口
2. 点击「下载结果视频」保存到本地
3. 点击「处理新视频」可以开始新的换脸任务

## 注意事项

- 视频时长建议不超过 5 分钟
- 人脸照片建议使用正面、光线充足的照片
- 处理时间取决于视频长度和人脸数量
- 结果视频会在服务器保存 7 天，请及时下载

## 常见问题

### Q: 为什么检测不到人脸？
A: 请确保视频中有清晰的人脸画面，人脸过小或模糊可能无法检测。

### Q: 换脸效果不理想怎么办？
A: 尝试使用更清晰的人脸照片，或勾选「HD 人脸增强」选项。

### Q: 处理失败怎么办？
A: 请检查视频格式是否正确，或尝试重新上传。如问题持续，请联系技术支持。
