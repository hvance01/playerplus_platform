# Prompt 管理功能使用说明

## 功能介绍

Prompt 管理功能用于创建、编辑和管理 AI 提示词模板，帮助您快速生成高质量的 AI 文案。

## 功能状态

此功能正在开发中，敬请期待。

## 计划功能

### Prompt 模板管理

- 创建自定义 Prompt 模板
- 编辑和删除现有模板
- 模板分类管理
- 模板收藏功能

### 变量占位符

- 支持在模板中使用变量占位符
- 使用时自动提示填写变量
- 变量类型支持：文本、数字、选项

### 模板分享

- 导出模板为 JSON 格式
- 导入他人分享的模板
- 团队模板共享

## 使用场景

- 营销文案生成
- 产品描述撰写
- 社交媒体内容创作
- 客服回复模板
- 邮件模板
