### 2.1 云服务商对比

我们调研了 10+ 个云服务商，基于项目核心需求进行对比：

结论：Railway 在成本、部署效率、运维负担三个核心维度均表现最佳，是 MVP 阶段的最优选择。

### 2.2 部署架构

前后端分离部署方案：

架构图：

为什么前端不放 Railway：

React 应用构建后是纯 HTML/CSS/JS 静态文件，不需要 Node.js 运行时

Railway 部署静态站点需要额外跑一个 nginx 容器，增加 $2-5/月成本

Cloudflare Pages 免费且自带全球 CDN，国内用户访问延迟更低（~100ms vs ~300ms）

跨域处理：

前端域名：app.example.com（Cloudflare Pages）

后端域名：api.example.com（Railway 自动分配或自定义）

后端 Gin 配置 CORS 允许前端域名即可
